# ☁️ CloudSphere

A modern **Cloud Computing Static Website** developed using **HTML, CSS, and JavaScript** and deployed using **GitHub Pages**.

---

## 📌 Project Overview

CloudSphere is a responsive static website that demonstrates cloud computing concepts through a modern web interface. It showcases cloud services, cloud architecture, deployment models, pricing plans, and a contact page.

This project is designed as part of a **Cloud Computing** course to demonstrate **Static Website Hosting** on the cloud.

## 🌍 Deployment

This project is deployed using **GitHub Pages**, which provides free cloud hosting for static websites.

Deployment Steps:

1. Create a GitHub Repository.
2. Upload all project files.
3. Open Repository Settings.
4. Navigate to **Pages**.
5. Select **Deploy from Branch**.
6. Choose **main** branch and **/(root)** folder.
7. Save the settings.
8. GitHub generates a live website URL.
